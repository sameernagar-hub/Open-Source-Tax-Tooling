# Deck Workspace

Use this folder for the presentation outline, slide source files, exported deck, speaker notes, and demo timing notes.

The deck should track the same argument as the report: research question, method, landscape, comparison, prototype, recommendation, and next steps.

- [`open_source_tax_tooling_draft_deck.pptx`](open_source_tax_tooling_draft_deck.pptx) is the Day 29 editable draft deck built from the outline and full report.
