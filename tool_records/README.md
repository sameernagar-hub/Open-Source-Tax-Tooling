# Tool Records

Use this folder for one structured evaluation record per tool.

Start each record from `template.md`, then rename the copy with a stable lowercase file name such as `beancount.md`, `ledger_cli.md`, or `firefly_iii.md`.

Each record should link to supporting source material, command output, screenshots, metadata snapshots, or notes in `evidence/`.
